## Descripción
Repositorio que contiene todos los laboratorios y asignaturas del curso Topicos Avanzados de Ingenieria de Software I. Incluye ejercicios practicos sobre principios de diseno de software, patrones de diseno (GoF), antipatrones, principios SOLID y refactorizacion de codigo.
